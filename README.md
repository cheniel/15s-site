# Hack Dartmouth Site #

* Fork the repo

* Install dependencies on virtualenv
```
$ pip install gdata
```
```
$ pip install Flask
```
```
$ pip install Flask-WTF
```
* Cd into the HackDartmouth directory
```
$ python run.py 
```
* Access the site in your browser on your local host port 5000 - http://127.0.0.1:5000/)
